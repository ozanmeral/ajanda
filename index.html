GitHub Bağlantı Kurulumu

Panele telefonunuzdan güvenli şekilde veri kaydetmek için bir defalık GitHub Erişim Anahtarınızı (PAT) girin. Bu anahtar sadece kendi cihazınızda saklanır.

💡 PAT Anahtarı Nasıl Oluşturulur:

  1. GitHub hesabınızda Erişim Anahtarı Ayarları sayfasına gidin.
  2. Anahtara bir isim verin (Örn: `Care Tracker`). Süresini "No expiration" (Süresiz) yapın.
  3. Repository access kısmından ilgili deponuzu seçin.
  4. Permissions altında, Repository permissions sekmesini açıp Contents yetkisini Read and Write (Oku ve Yaz) yapın.
  5. En alttaki Generate token butonuna tıklayarak oluşturulan kodu kopyalayıp buraya yapıştırın!
